# ジャンケン戦略バトル～ジャンバト～

## 制作LPの説明
- ランダムで配られた手札を使って、ジャンケンバトルをします
- 相手の手札と見比べて、勝ちやすい手を選びステージクリアを目指します

## 工夫した点・こだわった点
- ランダムでじゃんけんの手と攻撃力、効果を組み合わせて作成して、ユーザーが戦略を立ててプレイできるようにしました
- カードごとに特殊効果をつけることができたので戦略性に幅が出たことと、ステージが進むと難易度が上がることをどうにか機能として盛り込めたのでよかったです

## 次回トライしたこと（または機能）
- 本当は、強いカードを手に入れて山札を入れ替えて強くなっていく、敵のカード自体もだんだん強くなるようにしたかったです
- 山札と捨て札と手札をそれぞれ管理しており、下地までは構想ができてましたが時間的に断念しました
- また、デザインも現在は要素に色を付けたものをそのまま利用してますのでかなり無機質になってしまったかと思ってます
